# EvolvNex Client CRM

A modern, lightweight client relationship management (CRM) application built with [Next.js](https://nextjs.org/), [Supabase](https://supabase.com/), and [Tailwind CSS](https://tailwindcss.com/). It provides a clean dashboard for managing client data with secure authentication backed by Supabase.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Database & Auth**: Supabase
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

1. Copy `.env.example` to `.env.local` and fill in your Supabase project URL and anon key.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
